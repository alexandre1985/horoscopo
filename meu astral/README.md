Meu Astral
==========

Programa que lê o teu horoscopo com vários previsores.  
Actualmente tem previsores do Brasil (Eunice Ferrari), de Portugal (Maya, Paulo Cardoso, etc.) e Internacionais (Adrian Duncan). Isto só para nomear alguns.

Vai buscar a informação à pagina do [SAPO Lifestyle Astral](http://lifestyle.sapo.pt/astral/previsoes/).

## Plataformas
Está disponível para as seguintes plataformas: Windows, Mac, Linux, etc.

## Instalar
* Windows: clica duas vezes no instalador, que ele instala tudo automaticamente.
* Mac: faz duplo clique para descomprimir o ficheiro .zip, e depois faz copia o ficheiro .app para a pasta Aplicações
* Linux: faz unzip e executa o script `install` ou o script `install_root`
